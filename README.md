# Alkaline Richlyn.org 💰✨

A luxury-style investment website built with HTML, CSS, and JavaScript.  
Designed with a black & gold fintech theme for a premium financial experience.

---

## 🚀 Project Overview

**Alkaline Richlyn.org** is an investment platform interface that includes:

- User registration & login pages
- Deposit and withdrawal pages
- User dashboard with balance display
- Admin panel (basic structure)
- Modern fintech UI (black & gold design)
- Mobile responsive layout

---

## 🎨 Features

- 💳 Deposit system UI (MTN Mobile Money integration design)
- 💸 Withdrawal page interface
- 📊 User dashboard with account balance
- 🔐 Authentication pages (Login / Signup)
- 🧑‍💼 Admin panel layout
- ⚡ Smooth animations & loading screen
- 📱 Mobile-friendly design
